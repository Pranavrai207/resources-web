Figma Variables map to CSS Custom Properties

Premium sites don't hardcode values. Learn to read Figma's Local Variables (Colors, Spacing, Radii) and mirror them perfectly in your `:root` CSS selector. This ensures design consistency and makes implementing Light/Dark modes mathematically perfect.

:root {
  --space-md: var(--figma-token-space-4);
  --bg-primary: var(--figma-color-surface);
}

Translating Smart Animate to GSAP Timelines

When a designer builds a scroll prototype or interactive component in Figma, they use bezier curves and timings. Your job is to inspect those specific easings in Figma's Dev Mode and recreate them using `gsap.timeline()` and `ScrollTrigger`.

Key Skill: Reading Figma prototype interactions and translating standard ease-in-out to GSAP's CustomEase or standard power2.inOut.

Fake 3D: Perspective & CSS Transforms

You don't always need Three.js. You can achieve stunning parallax and depth by taking layered Figma frames, wrapping them in a container with `perspective: 1000px`, and using GSAP to animate their `translateZ` and `rotateY` properties on scroll.

Key Concept: Export Figma layers as separate transparent PNGs/WebPs, stack them absolutely, and apply differential z-axis translation linked to ScrollTrigger.

The "Apple Effect": Image Sequence Scrubbing

To build scroll-driven 3D product reveals without WebGL, designers render out 100+ frames (images) in Figma or Blender. You then load these onto an HTML5 `` and use GSAP ScrollTrigger to update the drawn frame based on the user's scroll position.

  • Export high-res frames from Figma prototype flows.
  • Preload images via JavaScript arrays.
  • Use GSAP to animate an object containing the `frameIndex` property.
  • Redraw canvas on `onUpdate`.
